# Dynamic programming

复杂问题 -> 分解简单的子问题，用递归的方式。

和分治类似。差异是有最优子结构，中途要淘汰次优解。

分治 + 最优子结构。



动态规划关键点：

> 最优子结构
>
> 存储中间状态
>
> 递推公式（状态转移方程）



递归公式示例：

```js
Fib(n) = Fib(n - 1) + Fib(n - 2);

二维路径 opt(i, j) = opt(i + 1, j) + opt(i, j + 1);
```



动态规划的截图步骤：

1. 定义子问题
2. 状态数组，DP方程
3. merge 子问题
4. 递归 or bottom-up
